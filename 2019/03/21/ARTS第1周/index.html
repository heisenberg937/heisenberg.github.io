如何改变数据

vue的data object：

react的state object:

对一个数据：name:”Sunil”，

vue:引用：this.name;改变： this.name = ‘John’；

react: 引用：this.state.name, 改变：this.setState({name: ‘John’})

可以看出react在书写上要麻烦一些，这里有一段解释：

This is because React wants to re-run certain life cycle hooks, [such as] componentWillReceiveProps, shouldComponentUpdate, componentWillUpdate, render, componentDidUpdate, whenever state changes. It would know that the state has changed when you call the setState function. If you directly mutated state, React would have to do a lot more work to keep track of changes and what lifecycle hooks to run etc. So to make it simple React uses setState.

大意是：react想重复运行几个生命周期钩子，比如render等，当你调用setState函数时，react知道状态改变了。如果你直接操作状态，react就不得不做更多的工作来跟踪变化以及决定应该运行什么生命周期钩子函数。为了保持简单react使用了setState。

​ 最近项目用到一个开源在线演示工具reveal.js，基于它做个展示平台。最近刚开始着手，动手的时候看了一点点工具所提供的功能就开始动手写代码，遇到问题就在那里调试半天，找不出问题，怀疑这里怀疑那里，向实验室同学询问，同学有条不紊的帮我调试，跟踪问题十分熟练，然后去看工具包提供的api，仔细看其中的说明，找到了出问题的地方，使任务向前推进了一步。
